Markdown学习笔记

Markdown介绍

Markdown 是一种轻量级的标记语言，它允许使用易读易写的纯文本格式的编写文档。
Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。

常用的快捷键

  • 加粗：ctrl + B
  • 标题：ctrl + H
  • 插入链接：ctrl + K
  • 插入代码：ctrl +shift + C —无法使用
  • 行内代码：ctrl +shift + K
  • 插入图片：ctrl +shift + I

引用文字

> + 空格 + 引用文字

无序列表、有序列表和任务列表

  • 使用 * + - 都可以创建一个无序列表

    • AAA
    • BBB

  • 使用1. 2. 3. 就可创建一个有序列表

  • 使用\ -[]不勾选 \ -[X]勾选 创建一个任务列表

    • 不勾选
    • 勾选

代码块

插入程序代码的方式有两种：使用反引号`(~键)，使用缩进Tab

  • 插入行内代码：插入一个单词或者一句代码的情况，使用 code这样的形式插入。
  • 插入多行代码输入3个反引号``` + 回车，并在后面选择一个语言名称即可实现高亮。

数学公式

按下 $$，然后按下回车键，即可进行数学公式的编辑。

$$
x^2 = 25
$$

插入表格

输入 | 表头1 | 表头2| 并回车，快捷键 ctrl + T 弹出对话框

  • 第二行为分割表头与主题的部分，第三行开始每一行为一个表格行

  • 列与列之间用管道符号 | 隔开

  • 设置对齐方式（表头与内容之间），如果不使用对齐标记，内容默认左对齐，表头居中对齐

    1. 左对齐 :|
    2. 右对齐 |:
    3. 中对齐 :|:

例如：

用户名 邮箱 年龄 性别
张 123@qq.com 23 男

常用标记（加粗、斜体、下划线）

  • 斜体

    使用 *单个星号* 或者 _单下划线_ 可以字体倾斜。快捷键 Ctrl + I

  • 加粗

    使用 **两个星号** 或者 __两个下划线__ 可以字体加粗。快捷键 Ctrl + B

  • 加粗斜体

    使用***加粗斜体***可以加粗斜体。

  • 删除线

    使用删除线 ~ 快捷键 Alt + Shift + 5

  • 下划线

    通过<u>下划线的内容</u> 或者 快捷键Ctrl + U可实现下划线

表情符号
